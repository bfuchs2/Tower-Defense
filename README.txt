A tower defense style game where the player can drop turrets from the sky to stop the soldiers from reaching the other side of the board.

Almost all textures used are from the asset store.
Turret textures from Turrets Pack.
Explosions from Porgezomp Pyroclastics.
Crate textures from Metal Crates.
Weapon sounds from Futuristic Weapons Set.
Enemies from Soldiers Pack.

An executable file of the game can be found in the builds folder.
